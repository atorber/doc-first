# 代码库索引

索引代码库结构和常用代码模式。

## 代码结构

```
src/
├── components/       # React 组件
├── services/        # API 服务
├── utils/           # 工具函数
└── stores/          # 状态管理
```

## 常用模式

### 模式1：API 调用
```typescript
// 位置：src/services/api.ts
export async function fetchData(url: string) {
  const response = await fetch(url);
  return response.json();
}
```

### 模式2：组件定义
```tsx
// 位置：src/components/Example.tsx
export const Example: React.FC = () => {
  return <div>...</div>;
};
```
