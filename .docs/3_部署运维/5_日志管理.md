# 5. 日志管理

## 5.1 日志级别

- **DEBUG**: 详细调试信息
- **INFO**: 一般信息
- **WARN**: 警告信息
- **ERROR**: 错误信息

## 5.2 日志存储

- **本地日志**: `logs/app.log`
- **集中存储**: ELK Stack / Loki
- **保留期限**: 30 天

## 5.3 日志查询

```bash
# 查看最新日志
tail -f logs/app.log

# 搜索错误日志
grep ERROR logs/app.log

# 使用 ELK 查询
# 在 Kibana 中查询日志
```
