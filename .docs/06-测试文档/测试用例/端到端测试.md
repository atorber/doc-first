# E2E 测试用例

## 场景：用户登录流程

**步骤**：
1. 打开登录页面
2. 输入用户名和密码
3. 点击登录按钮
4. 验证跳转到首页

**测试代码**：
```javascript
test('用户登录', async ({ page }) => {
  await page.goto('/login');
  await page.fill('#username', 'test');
  await page.fill('#password', '123456');
  await page.click('button[type="submit"]');
  await expect(page).toHaveURL('/home');
});
```
